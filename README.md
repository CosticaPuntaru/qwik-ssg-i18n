# Qwik Speak ⚡️

> Internationalization (i18n) library to translate texts, dates and numbers in Qwik apps.

Live example on [StackBlitz](https://stackblitz.com/edit/qwik-speak)

## Development Builds
### Server-side Rendering (SSR) and Client
```Shell
npm start
```
### Test
```Shell
npm test
```

## Production Builds
```Shell
npm run build
```
